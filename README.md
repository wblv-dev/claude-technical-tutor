# Claude Technical Tutor

Built this to turn Claude Code into a guided discovery tutor, primarily to help myself learn by doing rather than being told. I'm a constructivist learner — knowledge sticks when I research, build, and process it myself. Claude researches vendor docs, points me to specific sources, and guides me through building hands-on. It critiques my understanding rather than presenting answers, and captures structured notes as I work so the things I learn don't evaporate when the session ends. Hopefully this helps someone else with a similar brain.
