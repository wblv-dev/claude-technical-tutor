# Claude Technical Tutor

Built this to turn Claude Code into a technical training tool, primarily to help myself with explaining concepts, design choices, taking notes, exam prep etc. I'm a hands-on and visual learner and I'm bad at remembering to take notes. I'd start building things in the lab or working on concepts for my CISSP exam and then come back a few weeks later only to realise I've missed key decisions or notes from my Obsidian vault and get quite annoyed with myself about it, hopefully this will help someone else with a similar brain.
